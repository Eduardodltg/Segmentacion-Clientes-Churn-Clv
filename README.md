# Segmentacion-Clientes-Churn-Clv (Python)

## 📌 Propósito
Aplicar técnicas de aprendizaje no supervisado para segmentación de clientes y análisis de valor.

## 📂 Datos
- Dataset: Información de clientes (edad, ingresos, historial de compras, frecuencia, etc.)

## 🔧 Técnicas aplicadas
- Estadística descriptiva para explorar la distribución de variables
- Algoritmos de clustering: K-Means y DBSCAN
- Modelos de predicción de abandono (churn) y estimación de Customer Lifetime Value (CLV)

## 📊 Resultados
- Se identificaron segmentos de clientes con patrones de compra diferenciados.
- El modelo de churn permitió identificar clientes en riesgo de abandono.
- Estimación del valor del cliente en el tiempo.

## ✅ Conclusiones
La segmentación y el análisis de CLV ayudan a diseñar estrategias de retención y marketing personalizado.
